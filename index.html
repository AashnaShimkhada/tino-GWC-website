Introduction

Cupertino Girls Who Code is now a nationally registered club. This club's primary goal is to provide girls exposure to coding and computer science as well as build a strong community between all of our club members.

Let's close the gender gap in technology, together.

Java/Python

We work on projects that include Java and Python to create programs of your choice. Using knowledge of Java, we also develop games that range from basic to complex intensity.

Web Development

During our weekly meetings, we learn the basics of HTML and CSS to create websites and display information. Girl Coders from beginning to expert skills are all welcome!

National Curriculum

We use the national resources provided by Girls Who Code. These resources allow for Tino Girl Coders to learn from a variety of lessons and activities.
